### Vim plugins

To setup vim plugins, run this:

```sh
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
cp ./vimrc ~/.vimrc
```

Then run `PluginInstall` command in `vim`
